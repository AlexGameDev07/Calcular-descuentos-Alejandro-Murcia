HTML

  • ¿Qué es HTML y cuál es su función principal en el desarrollo web?


    HTML (HyperText Markup Language) es el lenguaje estándar para la creación de páginas web. Su función principal es estructurar el contenido de una página web mediante etiquetas, que permiten definir encabezados, párrafos, imágenes, enlaces y otros elementos.

  • ¿Qué son las etiquetas HTML y cómo se utilizan?


    Las etiquetas HTML son instrucciones que definen la estructura y contenido de una página web. Se escriben entre corchetes angulares (<>) y generalmente tienen una etiqueta de apertura y otra de cierre ().

  • Proporciona ejemplos de algunas de las etiquetas HTML más comunes.


    <body> para el contenido. <head> para información sobre el documento. <div> división dentro del contenido. <a> para enlaces. <strong> para poner el texto en negrita. <br> para saltos de línea. <H1…> <H6> para títulos dentro del contenido. <img> para añadir imágenes al documento.

CSS

  • ¿Qué es CSS y cómo se utiliza para dar estilo a una página web?


    CSS (Cascading Style Sheets) es un lenguaje de estilos que permite dar formato a una página web, definiendo colores, fuentes, márgenes y diseños de los elementos HTML.

  • ¿Cuáles son las propiedades más importantes de CSS?


    Algunas de las propiedades más utilizadas en CSS incluyen:
    color → Define el color del texto.
    background-color → Establece el color de fondo.
    font-size → Modifica el tamaño de la fuente.
    margin → Controla los márgenes exteriores.
    padding → Ajusta el espacio interno de un elemento.
    border → Define los bordes.
    width y height → Determinan el ancho y alto de un elemento.
    display → Controla la visibilidad y disposición de los elementos.

  • Explica el modelo de caja (box model) en CSS.


    El Box Model es un concepto fundamental en CSS que describe cómo se estructuran los elementos en una página web. Cada elemento se compone de cuatro partes:
    Contenido (content) → El área donde se muestra el texto o imágenes.
    Relleno (padding) → Espacio entre el contenido y el borde.
    Borde (border) → Línea que rodea el contenido y el relleno.
    Margen (margin) → Espacio entre el borde del elemento y otros elementos.

  • ¿Qué es el diseño responsivo y cómo se logra con CSS?


    El diseño responsivo permite que una página web se adapte a diferentes tamaños de pantalla (PC, tablet, móvil). Se logra mediante:
    Medidas flexibles (%, em, rem, vh, vw).
    Media queries (@media) para aplicar estilos según el tamaño de la pantalla.
    Flexbox y Grid para estructurar los elementos de forma flexible
